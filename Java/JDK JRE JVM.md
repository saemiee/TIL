# JDK JRE JVM

- JDK(개발환경)   
자바 개발도구의 약자(Java Development Kit)   
JDK = JRE + 개발을 위한 도구     
컴파일러, 디버그 도구 등이 포함  
버전과 종류가 있다.   
JDK는 Java SE, Java EE, Jave Me 3가지로 분류할 수 있다.
    - Java SE
        
        자바의 핵심으로 일반적으로 자바라고 하면 Java SE를 의미한다.
        
    - Java EE
        
        기업용 시장에서 사용하는 자바 개발환경
        
    - Java Me
        
        모바일 개발을 위해서 사용하는 자바 버전
        

- JRE  
자바 실행 환경의 약자   
(Java Runtime Environment)   
JRE = JVM + 자바 프로그램 실행에 필요한 라이브러리 파일 등   
JVM의 실행환경을 구현   
사용자를 위함   

- JVM
자바 가상 머신의 약자   
(Java Virtual Machine)
바이트코드를 읽고 검증하고 실행한다.   
Java의 버전 = JDK의 버전    
Java 를 설치하면 같이 설치가 된다.
