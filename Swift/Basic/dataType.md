## Data Type

Swift는 데이터 타입에 굉장히 엄격한 언어이기 때문에 타입을 명확하게 구분해야 한다.

- Bool
  true, false
- Int
  정수형
- UInt
  부호가 업는 정수, 0과 양의 정수를 다룬다.
- Float
  32bit 부동소수점 타입
- Double
  64bit 부동소수점 타입
- Character
  "한" 글자
- String
  "여러 문자"
- Any
  모든 타입을 지정
- AnyObject
  모든 클래스 타입을 지칭하는 클래스 타입
- nil
  엄밀히 말하면 타입은 아니다. 존재하지 않음을 나타내는 키워드로 타언어에서는 Null을 의미한다.
